1. Cursor or other softwares having the same function.
This sort of softwares will improve a lot when realizing the designed models and visualization. 
2. AI assistant like ChatGPT, Deepseek, Gork and so on.
**Do not use** these chat boxes to write code due to low efficiency: It is slow to manage a primary code when frequent adjustments are required.
3. Text Editor 
Word or overleaf (online latex editor)
4. Chart Maker
PPT or draw.io(a website to generate editable picture)
5. Resource
AI assistant's online search, Kaggle or essay searcher (https://www.semanticscholar.org)
6. Your brain and desire 
Tools are only used to be helpful. If you don't want to attend the competition, do not join or it will be a torture for your team.
